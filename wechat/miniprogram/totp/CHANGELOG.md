# Changelog

本文件遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 规范，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-06-28

### Added

- TOTP 安全码小程序首次发布
- 支持 TOTP（基于时间的一次性密码）生成与展示
- 支持添加、编辑、删除 TOTP 条目
- 支持按 issuer 和 username 分组管理
- 支持拖拽排序功能
- 支持微信登录与 access token 自动刷新
- 使用 Deno 作为包管理器
- 集成 TDesign 小程序组件库
